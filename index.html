Bagian 1: 🌐 Konsep DOM (Document Object Model)

📝 Apa itu DOM?

DOM adalah cara browser melihat dan mengorganisir halaman web. Bayangkan DOM seperti pohon keluarga dari semua elemen HTML di halaman web kita.

🎯 Penjelasan Sederhana DOM:

  • 📝 Document: Seluruh halaman web
  • 🧩 Object: Setiap elemen HTML/tag HTML (seperti button, input, div)
  • 🌳 Model: Cara browser mengatur semua elemen tersebut seperti pohon

🔍 Mengapa DOM Penting?

DOM memungkinkan JavaScript untuk:

  • 👁️ Melihat semua elemen HTML di halaman
  • 🔄 Mengubah isi dan tampilan elemen
  • 📞 Merespons ketika user mengklik atau mengetik
  • ✨ Membuat halaman web yang interaktif!

🌍 Analogi DOM - Rumah dan Isinya:

Bayangkan halaman web seperti rumah:

  • Document = Seluruh rumah
  • Objects = Setiap ruangan (kamar, dapur, ruang tamu)
  • Model = Denah rumah yang menunjukkan di mana setiap ruangan

JavaScript bisa "masuk" ke rumah dan mengubah isi setiap ruangan!

Bagian 2: 🎯 Mengambil Elemen dengan ID

🔍 Cara JavaScript "Mencari" Elemen

Untuk bisa mengubah sesuatu, JavaScript harus dulu "menemukan" elemen yang ingin diubah. Cara termudah adalah menggunakan ID.

Syntax untuk mengambil elemen:
document.getElementById("nama-id")

🎮 Analogi - Mencari Teman dengan Nama:

ID seperti nama unik setiap orang. Ketika kamu panggil "Budi!", hanya Budi yang merespons. Begitu juga dengan ID di HTML!

🧪 Latihan: Mengambil Elemen

Input nama:

Nama yang diambil: -

Kode yang bekerja di belakang layar:
// Langkah 1: Cari elemen input dengan id "namaUser"
let inputElement = document.getElementById("namaUser");

// Langkah 2: Ambil nilai yang diketik user
let nama = inputElement.value;

// Langkah 3: Tampilkan di elemen dengan id "tampilNama"
document.getElementById("tampilNama").innerHTML = nama;

📊 Mengambil Nilai Input dengan .value

Untuk elemen input, kita gunakan .value untuk mengambil apa yang diketik user.

let nilaiInput = document.getElementById("nama-id").value;

Bagian 3: 🏷️ Method innerHTML - Mengubah Isi Elemen

📝 Apa itu innerHTML?

Sekarang setelah kita tahu cara "menemukan" elemen dengan DOM, kita belajar cara mengubah isinya. innerHTML adalah method DOM untuk mengubah atau mengambil isi dari elemen HTML.

🎯 Penjelasan Sederhana:

Bayangkan innerHTML seperti kotak ajaib yang bisa:

  • 📖 Membaca apa yang ada di dalam kotak (elemen HTML)
  • ✏️ Menulis atau mengubah isi kotak tersebut
  • 🧹 Menghapus isi kotak dan menggantinya dengan yang baru

📚 Cara Menggunakan innerHTML:

1. Mengambil isi elemen:
let isi = document.getElementById("namaId").innerHTML;
2. Mengubah isi elemen:
document.getElementById("namaId").innerHTML = "Isi baru";

🧪 Latihan Praktis innerHTML:

Klik tombol untuk melihat innerHTML bekerja:

Ini adalah isi asli kotak
Contoh kode untuk tombol "Ubah Isi":
document.getElementById("kotakAjaib").innerHTML = "<strong>Isi sudah diubah!</strong>";

🎨 innerHTML dengan HTML Tags:

Kekuatan innerHTML: bisa menambahkan elemen HTML lengkap!

Isi akan muncul di sini...
Contoh: innerHTML dengan HTML tags
let daftar = "<ul><li>Item 1</li><li>Item 2</li></ul>";
document.getElementById("kotakHTML").innerHTML = daftar;

Bagian 4: ⚖️ Perbedaan .value dan .innerHTML

🤔 Kapan Gunakan .value vs .innerHTML?

Sekarang kita sudah tahu cara mencari elemen dan mengubah isinya. Tapi kapan pakai .value dan kapan pakai .innerHTML?

Gunakan .value Gunakan .innerHTML
• Input text
• Input number
• Textarea
• Select dropdown 		• Div
• Span
• Paragraph (p)
• Heading (h1, h2, dll)

🚨 Perbedaan Penting:

  • .value: Untuk elemen yang bisa diketik user (input)
  • .innerHTML: Untuk elemen yang menampilkan konten
  • innerHTML bisa menambahkan HTML tags seperti <b>, <i>, <br>
  • .value hanya berisi text biasa (tidak bisa HTML tags)

🧪 Latihan: Bandingkan .value vs .innerHTML

Test .value (untuk input):
Hasil akan muncul di sini
Test .innerHTML (untuk div):
Klik saya untuk ubah dengan innerHTML
Kode untuk demo di atas:
// Untuk .value:
let text = document.getElementById("testInput").value;
document.getElementById("hasilValue").innerHTML = text;

// Untuk .innerHTML:
document.getElementById("testDiv").innerHTML = "<strong>Diubah dengan innerHTML!</strong>";

Bagian 5: 🧮 Praktik: Membuat Kalkulator Sederhana

🎧 Sekarang kita terapkan semua yang sudah dipelajari:

  • ✓ DOM untuk mencari elemen input dan output
  • ✓ .value untuk mengambil angka dari input
  • ✓ .innerHTML untuk menampilkan hasil

🔍 Penjelasan Kode Kalkulator:

Langkah 1: Mengambil nilai input

let a = Number(document.getElementById("angka1").value);
let b = Number(document.getElementById("angka2").value);

Penjelasan: Kita ambil nilai dari kedua input dan ubah menjadi angka dengan Number()

Langkah 2: Melakukan operasi matematika

let hasil = a + b; // untuk penjumlahan

Penjelasan: Kita lakukan operasi sesuai tombol yang ditekan (+, -, ×, ÷)

Langkah 3: Menampilkan hasil

document.getElementById("hasil").innerHTML = hasil;

Penjelasan: Kita tampilkan hasil ke elemen dengan id "hasil" menggunakan innerHTML
